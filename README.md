# Distributed Vehicle Reservation Management System (DVRMS)

Course: COMP 6231 â€“ Distributed Systems

## Overview
Distributed vehicle reservation system with three office servers (MTL, WPG, BNF).
Client-server uses Java RMI; inter-server uses UDP.

## Features
- Role separation: Manager / Customer
- Vehicle inventory with quantity support
- Waiting list with automatic assignment
- Cross-office search and remote reservation with office constraints
- Budget enforcement
- Server-side concurrency control
